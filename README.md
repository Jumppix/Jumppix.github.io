## TikTok-Verifizierung mit URL Prefix

1. Lade die von TikTok bereitgestellte Signature-Datei herunter.
2. Lege sie im Repo-Root ab, also neben `index.html`, `privacy-policy.html` und `terms-of-service.html`.
3. Committe und pushe die Datei in `main`.
4. Warte kurz, bis GitHub Pages neu deployed hat.
5. Prüfe, ob die Datei unter `https://jumppix.github.io/<dateiname>` erreichbar ist.
6. Klicke in TikTok auf `Verify`.
